Signing Up for an Account

To sign up for a Nimbul user account, complete these steps.

  1. In the main Nimbul menu, click New Account. (If you are currently logged in to Nimbul, the New Account option will not appear. Log out first if you want to create a new account.)
  1. Fill in the requested information:
Field Description
Name Enter a display name (optional).
User Name Enter the name you will use as your login name (it can be the same as your display name). Must be at least 3 characters (max 40). Permitted characters: letters, numbers, periods, hyphens, underscores and “at” symbols.
E-Mail Your account activation message will be sent to this address.
Password Must be at least 6 characters (max 40). Use a combination of letters and numbers.
Time Zone Select the time zone from which you will be performing tasks in Nimbul. (Dates and times are used throughout the system, so be sure to select the correct time zone.)
Public Key Enter the public key from your Amazon EC2 key pair. If you do not enter a key in this field, you will still be able to access instances through the Nimbul UI, but you will not be able to access instances via SSH or use the command-line tools.

For more information about EC2 key pairs, see the Amazon EC2 documentation.

  1. Click Sign Up.

You will receive an e-mail message that contains an account activation link. If you do not receive this e-mail, contact a Nimbul administrator.

After your account has been activated, a Nimbul administrator will need to configure your account permissions so you can launch and terminate instances and perform other operations. For more information, see Permissions and Roles.

Once your account has been activated, you can manage your profile, password and other account information by clicking [Your User Name]’s Account in the main Nimbul menu.
