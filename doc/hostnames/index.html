Hostnames and Elastic IP Addresses: Overview

Nimbul helps you manage hostnames and elastic IP addresses for your instances. Nimbul administrators can also expose services such as DynDNS to the infrastructure, using the Services area of the Admin Controls.

Role/Access Level Relevant Topics
All Users* Managing Hostnames
Managing IP Addresses

*To add a hostname or an address to a provider account, you must have administrator access to that account. Cluster administrators can assign hostnames to servers within the cluster, and can manage hostname leases and IP addresses for instances running under that server.
