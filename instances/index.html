Viewing and Updating Instances

*In order to stop, start, view and update instances, you must have administrator access to the appropriate server clusters. If an administrator has granted you SSH access based on a server profile within the cluster, you can still log in to an instance via SSH and restart it outside of Nimbul. See Setting Up SSH Access.

If you do not have the cluster access you need, contact your Nimbul administrator.
